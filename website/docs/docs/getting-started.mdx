## Usage

An accordion is created by wrapping any number of `AccordionItem` components inside an `Accordion` component.

By default, only one accordion item can be expanded at one time.

<Basic />

:::tip

When focus is on an accordion header, you may use the `Arrow Down` and `Arrow Up` keys
to move focus to the next or previous accordion header.

:::

<CodeBlock language="jsx">{transformCodeBlock(sourceBasic)}</CodeBlock>

Or, if your data is already in an array:

<CodeBlock language="jsx">
  {transformCodeBlock(sourceArrayMap)}
</CodeBlock>

:::info

Although the `header` prop of `AccordionItem` looks like some simple text content,
you may provide it with any JSX elements or React components,
allowing the item header to be [freely customised](./customising-header).

:::

## Accessibility

This library implements WAI-ARIA roles, states, properties, and keyboard interaction,
which is compliant with the [Accordion Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/).
